The detailed instructions needed to install this software have not yet
been written.
But at minimum you will need the following:

  - Lua version 5.1 or 5.2
  - The `luarocks` tool for downloading Lua packages.
  - The LYAML binding for YAML.  Install with `luarocks install lyaml`
  - A TeXLive distribution of 2024 or later.

Both Lua and TeXLive should be easy to install with your
distribution's package manager.

You'll want to put the `bin` directory on your $PATH.
